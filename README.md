# dice-roller

Roll dice. Pass dice and modifier and get a result. Written mostly for experience with node.

## Commands

* `npm install`
* `npm start`

Or, for auto-reloading use `nodemon`:

* `npm install nodemon --global`
* `nodemon server.js`
